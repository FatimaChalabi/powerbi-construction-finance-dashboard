## 1. Core Financial Metrics

```dax
Total Revenue = SUM(fact_financials[ActualRevenue])

Total Cost = SUM(fact_financials[ActualCost])

Total Profit = SUM(fact_financials[GrossProfit])

Gross Margin % = DIVIDE([Total Profit], [Total Revenue])

Total Projects = COUNTROWS(dim_projects_v2)

Planned Revenue = SUM(fact_financials[PlannedRevenue])

Planned Cost = SUM(fact_financials[PlannedCost])
```

---

## 2. Variance Analysis

```dax
Revenue Variance % = 
DIVIDE(
    [Total Revenue] - [Planned Revenue],
    [Planned Revenue]
)

Cost Variance % = 
DIVIDE(
    [Total Cost] - [Planned Cost],
    [Planned Cost]
)

Total Cost Overrun = 
SUM(fact_financials[ActualCost]) - SUM(fact_financials[PlannedCost])

Overrun Projects = 
CALCULATE(
    COUNTROWS(dim_projects_v2),
    FILTER(
        dim_projects_v2,
        CALCULATE(SUM(fact_financials[ActualCost])) >
        CALCULATE(SUM(fact_financials[PlannedCost]))
    )
)
```

---

## 3. Profitability Metrics

```dax
Avg Gross Margin % = 
AVERAGEX(
    dim_projects_v2,
    CALCULATE([Gross Margin %])
)
```

## 4. What-if Simulation

```dax
-- Cost Reduction % parameter auto-generated by Power BI What-if

Simulated Profit = 
[Total Revenue] - (
    [Total Cost] * (1 - 'Cost Reduction %'[Cost Reduction % Value] / 100)
)

Sim Margin % = 
DIVIDE([Simulated Profit], [Total Revenue])
```
